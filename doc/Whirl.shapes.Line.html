Class: Line

Whirl.shapes. Line

Represents a line in space defined by two pairs of x-y coordinates representing both end-points of the line.

Can be used for things such as ray-tracing, distance calculations, sight-line checks, etc.

Lines are constructed using the Whirl.shapes.Line factory method, but the underlying class can be accessed with Whirl.shapes.Line._class.

new Line( [x1] [, y1] [, x2] [, y2])

Parameters:
Name Type Argument Default Description
x1 number | Whirl.shapes.Point <optional>
 0

X-coordinate of the starting point. If giving an instance of a Point the y1 parameter should also be a Point.
y1 number | Whirl.shapes.Point <optional>
 0

Y-coordinate of the starting point. If both x1 and y1 are instead instances of a Point then the starting point is defined by the first given Point, and the ending point is defined by the second.
x2 number <optional>
 0

X-coordinate of the ending point.
y2 number <optional>
 0

Y-coordinate of the ending point.
Examples
Whirl.shapes.Line(50, 50, 100, 100); // Line {x1: 50, y1: 50, x2: 100, y2: 100}
Whirl.shapes.Line(
	Whirl.shapes.Point(50, 50),
	Whirl.shapes.Point(100, 100),
); // Line {x1: 50, y1: 50, x2: 100, y2: 100}

<readonly> vertices :Array.<Whirl.shapes.Point>

Returns an array of Points representing the two vertices of this line. The first Point represents the starting point of the line, and the second Point represents the ending point.
