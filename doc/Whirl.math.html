<static> map(value, in_min, in_max, out_min, out_max)

Extrapolate how far through a given range of values a point is to another given range of values.

For example, five (5) is fifty percent (50%) of the way between zero (0) and ten (10). To map that given point linearly between the range zero (0) to ten (10) to the range fifty (50) to one-hundred (100) would mean going fifty percent (50%) of the way between fifty (50) and one-hundred (100) to get seventy-five (75).

Parameters:
Name Type Description
value number

Value to use as a point between the first given range.
in_min number

Lower bound of the input range.
in_max number

Upper bound of the input range.
out_min number

Lower bound of the output range.
out_max number

Upper bound of the output range.
Returns:
Type
number
Example
Whirl.math.map(5, 3, 7, 50, 100); // 75
Whirl.math.map(25, 0, 50, 100, 200); // 150
Whirl.math.map(75, 50, 100, 1000, 2000); // 1500
Whirl.math.map(5, 10, 20, 100, 200); // 50

<static> random( [a] [, b])

By default, returns a random float in the range of zero (0) to one (1) (inclusive of zero (0) but not of one (1)).
If one argument is given then return an integer in the range of zero (0) up to the given value.
If two arguments are given then will return an integer in the range between the two given arguments.

Parameters:
Name Type Argument Description
a number <optional>

If given without b being given, will be the upper bound of the range. If given with b, will be the lower bound of the range.
b * <optional>

Upper bound of the range of the random number.
Returns:
Type
number
Example
Whirl.math.random(); // 0.6019569996537999
Whirl.math.random(); // 0.2947829307200911

Whirl.math.random(5); // 0
Whirl.math.random(5); // 4
