<static> randHSL( [cfg])

Generate a random HSL (Hue, Saturation, Lightness) value (hsl(h, s, l)).

Parameters:
Name Type Argument Description
cfg Object <optional>

Optional configuration when generating the HSL value.

Properties
Name Type Argument Description
alpha number <optional>

Add an alpha value to the generated HSL value (hsl(h, s, l, a)).
json boolean <optional>

Return a JSON object representation of the HSL value instead of a string.
sat number <optional>

Override the randomly generated saturation value.
lit number <optional>

Override the generated lightness value with the given value.
Returns:

Returns an object if cfg.json is true, otherwise returns a string.

Type
string | Object
Example
Whirl.util.randHSL(); // "hsl(39, 3%, 7%)"
Whirl.util.randHSL(); // "hsl(48, 85%, 78%)"
Whirl.util.randHSL({alpha: true}); // "hsla(85, 15%, 67%, 0.8)"
Whirl.util.randHSL({json: true}); // {hue: 177, sat: 20, lit: 27}
Whirl.util.randHSL({alpha: true, json: true}); // {hue: 174, sat: 66, lit: 40, a: 0.8}
Whirl.util.randHSL({json: true, sat: 80}); // {hue: 169, sat: 80, lit: 60}
Whirl.util.randHSL({json: true, lit: 45}); // {hue: 339, sat: 52, lit: 45}
